注意事项

  • 控制码的输入没有顺序，没有数量限制，但是三种控制码同时间内只能生效最靠右的一个，三种控制码的排序互相之间没有影响。

  • 控制码为对整个命令行的渲染，不被覆盖的话会一直存在（关掉终端再打开除外）。记得重置，即为用“0”这一标记值覆盖。
